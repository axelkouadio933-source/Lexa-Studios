# Lexa Server\n\n## Installation locale\n\n1. Ouvre un terminal dans ce dossier.\n2. Lance `npm run admin:setup` pour choisir ton identifiant et ton mot de passe admin.\n3. Lance `npm start`.\n4. Ouvre `http://127.0.0.1:4173`.\n5. L'admin privé est sur `http://127.0.0.1:4173/admin`.\n\n## Sécurité\n\n- L'admin n'est servi qu'après session serveur valide.\n- La session utilise un cookie `HttpOnly` et `SameSite=Strict`.\n- Le mot de passe est hashé avec `scrypt` dans `data/admin-auth.json`.\n- Les liens externes passent par `/go` avec liste de domaines autorisés.\n- Les clics de téléchargement sont journalisés dans `data/usage-events.jsonl`.\n\nPour publier sur Internet, utilise HTTPS devant Node, par exemple avec un proxy comme Nginx, Caddy, Cloudflare Tunnel ou un hébergeur Node.\n
